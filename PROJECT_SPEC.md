# 政治家ほっぺぷにぷにゲーム (PrimeTap) プロジェクト仕様書

## 1. プロジェクト概要

- **テーマ**: 政治家育成タップゲーム

- **デザイン基調**: 水色（Light Blue / Cyan）を基調とした、清潔感と透明感のある「グロッシー（光沢感）」デザイン。

- **主要メカニクス**: タップによるポイント稼ぎ、国家予算（コイン）によるアイテム収集、政治家との親密度向上。

## 2. データベース構造 (Data Schema)

### A. ユーザー (User Profile)

| フィールド名 | 型 | 説明 |
| --- | --- | --- |
| `name` | String | ユーザー名 |
| `homeCountry` | String | 所属国家 |
| `totalTaps` | Int | 全期間の累計タップ回数 |
| `totalPoints` | Int | 全期間の累計獲得ポイント |

| `budgetCoins` | Double | 国家予算（ガチャやアンロックに使用するコイン） |

| フィールド名 | 型 | 説明 |
| --- | --- | --- |
| `id` | String | 一意識別子 |
| `name` | String | 政治家名 |
| `country` | String | 所属国家 |
| `rarity` | Enum | レアリティ（Low/Medium/High/Boss） |
| `odds` | Double | 換金倍率（コイン計算用） |
| `isUnlocked` | Bool | 解放済みフラグ |

| `politicianPoints` | Int | その政治家の累計獲得ポイント |
| `intimacyLevel` | Int | 親密度レベル (1～3) |

### C. アイテム (Game Item)

| フィールド名 | 型 | 説明 |
| --- | --- | --- |
| `id` | String | 全100種類の一意識別子 (policy_001〜100) |
| `name` | String | 政策名 |
| `efficiencyBoost` | Double | タップ効率の上昇率 (永続 +0.02) |
| `isOwned` | Bool | 取得済みフラグ（重複なし） |

## 3. ゲームロジック

### ① タップポイント計算式

1タップあたりの獲得ポイント（タップポイント速度）は、以下の式で動的に算出されます。

> **ポイント = min(floor(1 * (1 + 3.472 * 総タップ回数 / 10000) * (1 + 3.472 * 政治家ポイント / 1000)), 20)**

- **総タップ回数**: 10,000回をゴール目安とした段階的な成長。

- **政治家ポイント**: 個別の育成状況によるボーナス。

- **上限**: 1タップ最大20ポイント。

### ② レベルアップ

- **条件**: 政治家ごとに **200タップ** ごとにレベルが上昇（最大Lv3）。

- **変化**: レベルに応じて育成画面の顔画像が変化。

### ③ アンロック条件

- **対象**: 日本以外の各国首脳。

- **条件**: 「国家予算 50ポイント」かつ「日本首脳のレベル 3」。

### ④ ガチャシステム

- **コスト**: 100コイン / 1回。

- **当たり (10%)**: 未取得の政策アイテムをランダムに獲得。タップ効率が永続的に向上。

- **外れ (90%)**: 50コインを返却（残念賞）。

## 4. UI/UX 構成

1. **スタート画面**: タイトルロゴを表示。育成開始、記録確認、免責事項への導線。

1. **育成画面 (メイン)**:
  - 中央に政治家の顔画像（タップでバウンドアニメーション）。
  - 親密度レベルとタップポイント速度（速度）をリアルタイム表示。
  - 下部メニューから世界地図、マイ政治家、アイテムへ遷移。

1. **世界地図**:
  - スクロール可能な地図上に各国ピンを配置。
  - ピンタップでその国の政治家カタログを表示。
  - カタログから直接アンロックや育成画面への切り替えが可能。

1. **マイ政治家一覧**: 解放済みの政治家をリスト表示。

1. **アイテム画面**:
  - ガチャ実行タブ（回転アニメーション演出）。
  - コレクションタブ（取得済み政策の一覧）。

1. **記録画面**: 累計タップ数、累計ポイント、現在の効率などをダッシュボード形式で表示。

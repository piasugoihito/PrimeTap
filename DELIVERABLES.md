# Flutterゲームアプリ成果物概要

このプロジェクトは、**Flutter完結**で動作する軽量なローカルゲームアプリです。ゲーム内容は、キャラクターをタップし続けて**タップポイント**を貯め、一定条件でキャラクターを解放しながら成長させるシンプルなタップゲームです。保存処理は `shared_preferences` を用いた**完全ローカル保存**で構成しており、基本プレイはインターネット接続なしでも継続できます。

| 項目 | 実装内容 |
|---|---|
| 画面構成 | スタート画面 / プレイ画面の2画面構成 |
| プレイ | キャラクタータップ、揺れアニメーション、成長演出 |
| 強化 | 一定タップごとに獲得する強化アイテムを消費して、1タップあたりのポイントを上昇 |
| 解放 | 累計タップポイントに応じて新キャラクターを解放 |
| 記録 | 日次記録としてタップ回数、最高タップ速度、タップポイントを保存 |
| 可視化 | カレンダー表示、日・月・年単位のグラフ表示 |
| 通信 | 基本プレイはローカル完結、接続時のみ下部バナー広告を表示 |

## 主なファイル構成

| ファイル | 役割 |
|---|---|
| `lib/main.dart` | アプリ本体、2画面UI、プレイ画面、広告組み込み |
| `lib/game_models.dart` | ゲーム状態、キャラクター、日次記録モデル |
| `lib/game_repository.dart` | ローカル保存、タップ更新、アイテム獲得、強化処理 |
| `lib/game_controller.dart` | UI向け状態管理 |
| `lib/record_widgets.dart` | カレンダーと日・月・年グラフ表示 |
| `lib/ad_banner.dart` | ネット接続時のみ表示する下部バナー広告 |

## 実行方法

Flutter SDK が利用可能な環境で、プロジェクトディレクトリへ移動してから通常の Flutter 実行手順で起動できます。

```bash
flutter pub get
flutter run
```

## 広告に関する補足

現在は**公式テスト用 AdMob ID** を設定しています。本番公開時には、Android と iOS のアプリIDおよびバナー広告ユニットIDを**自身の AdMob 情報に差し替えてください**。

> Android Manifest と iOS Info.plist には、テスト用のアプリIDが設定済みです。
